# Blade-And-Sorcery-HP-Modifer

**HP Modifier** — это мод для игры *Blade & Sorcery*, который добавляет в игру удобный ползунок для настройки здоровья NPC. Вы можете выставлять значения **от 1 до 1000**, что позволяет изменять прочность врагов под любой стиль игры — например, устраивать затяжные, эпичные дуэли или, наоборот, делать врагов уязвимыми.

---

## ✨ Особенности
* **Гибкая настройка:** Изменяйте здоровье всем спавнящимся NPC в диапазоне от 1 до 1000 единиц.
* **Идеально для хардкора:** Сделайте бои намного дольше и реалистичнее.
* **Удобный интерфейс:** Ползунок интегрирован прямо в игровое меню.

> ⚠️ **Важно:** Для корректной работы мода требуется **последняя актуальная версия игры**.

---

## 🛠 Установка

1. Скачайте архив с модом.
2. Перейдите в корневую папку игры и найдите директорию для модификаций:
   `.../BladeAndSorcery_Data/StreamingAssets/Mods`
3. Распакуйте содержимое архива в папку **Mods**.

---

## 🎮 Как использовать

1. Запустите игру.
2. Откройте игровое меню и перейдите во вкладку **Mods**.
3. Найдите мод **HP Modifier** и настройте ползунок здоровья по своему вкусу перед спавном врагов.
